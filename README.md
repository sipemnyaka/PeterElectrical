# Peter Electrical Website

A responsive, mobile-first business website for Peter Electrical, a residential and commercial electrical contractor serving Mthatha and surrounding areas.

## Objectives

- Create a professional online presence
- Learn HTML, CSS and JavaScript
- Deploy to Netlify
- Later migrate to Azure Static Web Apps
- Use the project as a portfolio piece

## Technologies

- HTML5
- CSS3
- JavaScript
- Git
- GitHub
- Netlify
- Azure Static Web Apps (planned)
